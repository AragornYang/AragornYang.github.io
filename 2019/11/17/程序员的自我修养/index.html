程序员的自我修养

提升自己

目的

  1. 提高工作能力，能够独立处理日常工作
  2. 提升自我幸福感、满足感、成就感
  3. 能够有较好的发展，有更好的经济收入
  4. 能够靠自己为父母、为爱的人提供更好的生活

做法

善于学习、勤加思考、掌握方法、勤奋努力、持之以恒，同时要以更加认真负责的态度对待工作和生活。

  1. 数据结构、操作系统相关书籍
  2. 英语能力
  3. 搜索能力
    1）关键词构造：“JAVA XML 序列化”
    2）常见的开源网站多去逛逛
  4. 思维模式，写代码前要尽可能的熟悉项目的业务流程
  5. 编程习惯
    1）快捷键的使用，提高工作效率
    2）多写注释，注意规范化
    3）进行代码review
    4）多用邮件进行工作

记住

持续学习，坚持才是程序员最大的修养
